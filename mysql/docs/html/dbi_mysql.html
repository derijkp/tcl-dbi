dbi_interface ?objectName?

DESCRIPTION

The dbi_mysql command creates an object objectName that can be used to connect to mysql databases using its native API. If objectName is not provided, a name will be automatically generated and returned by the command The object created supports the dbi interface for accessing and querying databases.
