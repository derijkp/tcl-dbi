DESCRIPTION

The dbi_admin interface is designed for administrating SQL databases. It presents a generic api to create and drop databases.

THE DBI_ADMIN INTERFACE

objectName open database ?arg ...?
open the given 'database'. Extra arguments can (must) be given, depending on the backend used. database is an identifier of the database. What this is exactly, can depend on the implementation This is method the same as the one defined by dbi interface.
objectName create database
create a database identified by database
objectName drop
Drop the database currently open.
