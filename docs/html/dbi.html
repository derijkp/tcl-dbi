DESCRIPTION

The dbi package contains the definition of the dbi interface and some associated tools. The dbi interface is further described in interface_dbi. The package can be loaded using the commands "package require dbi".

TOOLS

dbi::info types
Returns a list of all packages loaded or available that start with dbi_ (and should provide a dbi backend)
